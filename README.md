# Valentin App

Monorepo for the Valentin application ecosystem.

## Project Structure

- valentin-widget: Android application with interactive home screen widgets.
- valentin-web: Next.js website for visual management and user interface.

## Technologies

- Android (Java/Kotlin)
- Next.js (React/TypeScript)
- Vanilla CSS

## Setup

Refer to the respective subdirectories for specific installation and execution instructions.
